AI-Powered Natural Language SQL Query Agent
Built an AI-powered SQL query agent in N8N that converts user questions in plain English into SQL queries and executes them in a PostgreSQL database.
Designed a workflow where users provide requirements in natural language, the agent analyzes the request, generates SQL queries, retrieves relevant data, and returns structured outputs.
Implemented context memory to improve query accuracy and handle follow-up questions efficiently.
Tools & Technologies: N8N, PostgreSQL, Supabase, AI Agents, SQL, Simple Memory
